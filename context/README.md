# Harshit Interview Prep Context

This folder is the stable handoff context for teaching Harshit Mehra toward Google Senior SWE interview readiness.

Use these files at the start of future coaching sessions:

- `startup_protocol.md`: what to load and how to begin each daily session.
- `teacher_profile.md`: how to teach Harshit and how to calibrate feedback.
- `roadmap.md`: current preparation phase, priorities, and what not to over-study.
- `session_workflow.md`: how a DSA, system design, or behavioral session should run.
- `templates.md`: reusable modeling templates for DSA, system design, and behavioral stories.
- `progress_log.md`: lightweight session notes and next actions.
- `teacher_subagent_prompt.md`: reusable prompt for a subagent that briefs the main teacher.

The core coaching rule is simple:

> Preserve depth, add speed, and do not let preparation become an infinite syllabus.
