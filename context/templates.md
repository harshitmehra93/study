# Interview Templates

## Graph Template

Use before every graph problem.

- Nodes
- Edges
- Source(s)
- Target / success condition
- Natural or reversed direction
- Traversal
- Visited meaning
- Failure condition
- Complexity

Key coaching question:

> What is the smallest graph model that proves correctness?

## DP Template

Use for maintenance and new DP problems.

- State
- Base cases
- Decision
- Combine
- Validity
- Memo key
- Trap

Useful framing:

> Decision / Remaining / Combine

## Backtracking Template

Use when the backtracking block begins.

- Path
- Choices
- Validity
- When to record answer
- How to undo choice

## System Design Template

Use for senior system design practice.

- Requirements
- Scale
- APIs
- Data model
- Architecture
- Deep dive
- Trade-offs
- Failure modes
- Metrics / Observability

## Behavioral Story Template

Use for senior SWE behavioral packaging.

- Situation
- Task
- Action
- Result
- Trade-off
- Learning

Senior signals to surface:

- ownership
- judgment
- communication
- scope
- ambiguity handling
- technical leadership
